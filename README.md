# 🚗 Car Rental System

A simple **Car Rental System** project developed using **Core Java** and **OOPs concepts**.  
This project helps manage cars, customers, and rental operations in a simple console-based application.

---

# 📌 Features

✅ Add Cars  
✅ View Available Cars  
✅ Add Customers  
✅ Rent a Car  
✅ Return a Car  
✅ Rental Information Display  
✅ Object-Oriented Design

# 🚘 Classes Explanation

## 1. Car.java
Stores car details such as:
- Car ID
- Car Name
- Price Per Day
- Availability

---

## 2. Customer.java
Stores customer details such as:
- Customer ID
- Customer Name

---

## 3. Rental.java
Handles rental operations:
- Customer rents car
- Rental duration
- Rental information

---

## 4. CarRentalSystem.java
Main business logic:
- Add cars
- Display cars
- Manage rentals

---

## 5. Main.java
Program execution starts from here.
